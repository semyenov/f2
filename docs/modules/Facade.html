    Simplified Federation API Facade

    This module provides a simplified, user-friendly API for common federation patterns. It abstracts away the complexity of Effect-TS for developers who want quick setup while still maintaining all the type safety and performance benefits.

    Example: Quick federation setup

    import { Federation } from '@cqrs/federation'

    const federation = await Federation.create({
      entities: [userEntity, productEntity],
      services: ['http://users:4001', 'http://products:4002']
    })

    // Start the federation
    await federation.start()
