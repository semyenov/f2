    Entity reference resolver with Effect-based error handling

    Called when Apollo Federation needs to resolve an entity from a reference. The reference contains the key fields that identify the entity uniquely.

    Type Parameters

    • TResult

      The complete entity type to be returned

    • TContext

      The GraphQL execution context type

    • TReference

      The reference type containing key fields

      • Parameters

        • context: TContext

        • info: GraphQLResolveInfo

          GraphQL execution info with field selection and metadata

        Returns Effect.Effect<TResult, EntityResolutionError>

        Effect resolving to the entity or EntityResolutionError

    Example

    const resolveUser: EntityReferenceResolver<User, AppContext, UserRef> =
      (ref, ctx, info) => pipe(
        ctx.userService.findById(ref.id),
        Effect.mapError(err =>
          ErrorFactory.entityResolution(
            `User ${ref.id} not found`, 'User', ref.id, err
          )
        )
      )
